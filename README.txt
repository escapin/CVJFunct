%%%%%%%%%%%%%%%%%%%%%%%%%%%% COMPILE %%%%%%%%%%%%%%%%%%%%%%%%%%%%

The project can be compiled either using eclipse or using
Apache Ant with the buildfile "build.xml"

To compile the project, call ant from the main directory:
lib\apache-ant-1.8.4\bin\ant.bat (windows)
./lib/apache-ant-1.8.4/bin/ant (linux)

This should compile the sources into "bin". To be able to compile it, you
have to have Java JDK 1.7 installed and within your environment variable
PATH (or set JAVA_HOME correctly).
